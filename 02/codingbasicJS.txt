Key Concept

In JavaScript:

Primitive types are passed by value 
(A copy of the value is passed)

Objects and arrays are passed by reference 
(A reference to the original object in memory is passed)

Letâ€™s break this down.

ðŸŸ¢ 2. Pass by Value (Primitive Types)

When you pass a primitive type     like number, string, boolean,   to a function,
the function gets a copy of the value.

âž¡ï¸ So, changing it inside the function does not affect the original variable.

function changeValue(x) {
  x = 100; // change inside the function
  console.log("Inside function:", x);
}

let num = 50;
changeValue(num);
console.log("Outside function:", num);



ðŸ”µ 3. Pass by Reference (Objects & Arrays)

Objects and arrays are not copied when passed to a function.
Instead, their memory reference (address) is passed â€” so if you modify the object inside the function, it changes the original.
